### Prerequisites

- Unix-like operating system (Linux, macOS)
- Bash shell

### Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/Mechmank/Shell-script.git
   ```

2. Navigate to the repository directory:
   ```bash
   cd Shell-script
   ```

   ```bash
   chmod +x script_name.sh
   ```

   ```bash
   ./script_name.sh
   ```
